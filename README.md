# PyTorch-Based-Linear-Regression-Implementation-with-Gradient-Descent
본 프로젝트는 PyTorch를 활용하여 선형 회귀(Linear Regression) 모델을 직접 구현한 실습입니다. nn.Linear와 같은 고수준 API를 사용하지 않고, 손실 함수 정의 → 역전파(backward) → 경사하강법을 통한 파라미터 업데이트까지 학습 전 과정을 직접 구성하는 것을 목표로 합니다.  키(cm)와 몸무게(kg) 간의 관계를 예제로 사용하여, 선형 모델이 데이터를 어떻게 학습하고 최적의 직선을 찾는지를 시각적으로 확인합니다.
